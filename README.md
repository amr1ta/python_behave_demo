# Python Page Object Model Framework from Scratch

## To Run
* Clone repo
* Open terminal and cd to repo
* pipenv install
* pipenv shell
* cd to any day's directory
* Run python -m unittest
* day4 - to generate HTML - python day4/reportGen.py
